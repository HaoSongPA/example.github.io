Typography

Typographic hierarchy

Typographic hierarchy is a way to make the text more readable. The main purpose is to show to the reader which information to focus on—which is most important and which is simply supporting the main points. Without a typographic hierarchy, every letter, every word, and sentence in a design would look the same.

Web Safe Fonts

Web safe fonts are common fonts.
Using a web safe font means the font would display correctly on every browser.

Font Properties

Font Family

The font-family property specifies the font for an element.

  • font-family: "Times New Roman", Times, serif;
  • font-family: Arial, Helvetica, sans-serif;
  • font-family: 'Courier New', monospace;
  • font-family: "Lucida Handwriting", cursive;
  • font-family: Copperplate, Papyrus, fantasy;

Font Weight

The font-weight property sets how thick or thin characters in text should be displayed.

  • font-weight: normal
  • font-weight: bold
  • font-weight: bolder
  • font-weight: lighter

Web Fonts

Web fonts offer creative freedom. Designers can match the font style to the website's theme and brand identity. They are essential tool for modern web design.

Use of Google Fonts

  1. Go to https://fonts.google.com/
  2. Click on a font you would like and click “Get Font”
  3. Copy the HTML code and paste it in the head of your html
  4. Copy the CSS classes and paste them into your stylesheet and use the classes where you want to use them!
